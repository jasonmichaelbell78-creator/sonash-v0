## Recommendations for Research Consumers

1. **Do not depend on AutoDream.** Build consolidation infrastructure as if it does not exist (aligns with contrarian #6 and G5 verdict).
2. **Recalibrate the token budget.** The system is already at 19,240 tokens, above the proposed 18K warning threshold. Either raise the threshold for 1M context or prioritize token reduction.
3. **Budget 3x the estimated reconciliation effort** for canonical-memory sync (23 files, not 7).
4. **Treat the 70% elimination and 70-80% completeness figures as qualitative.** Neither has a disclosed methodology. They are useful as directional guidance, not as measurement.
5. **The abandonment rate (C-205) should be the primary design constraint** for every new mechanism proposed, not a footnote risk.
