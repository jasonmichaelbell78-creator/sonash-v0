## v1 Requirements

### Storage & Data Layer

- [x] **STOR-01**: Zod schemas defined for all 5 JSONL files with write-time
      validation
- [x] **STOR-02**: JSONL-first architecture — AI writes structured JSONL as
      source of truth, markdown generated as view
- [x] **STOR-03**: Split JSONL storage — each file has independent schema
- [x] **STOR-04**: Three-tier completeness model — every record tagged
      full/partial/stub
- [x] **STOR-05**: Structured origin field — Zod-validated object replacing
      string source IDs
- [x] **STOR-06**: Read-time validation — consumers log warnings for malformed
      records

      function
- [x] **STOR-08**: hasField() helper — checks both null AND completeness_missing
- [x] **STOR-09**: Test fixtures for all 3 completeness tiers

- [x] **PIPE-03**: Auto-deferred-item creation from review JSONL writes
- [x] **PIPE-04**: Unified invocation tracker for all skills/agents
- [x] **PIPE-05**: Merged promotion script — detect recurrence, generate rules
- [x] **PIPE-06**: Automated rule generation from pattern recurrence
- [x] **PIPE-07**: CLAUDE.md auto-generation from top patterns

- [x] **PIPE-09**: 3 security-specific FIX_TEMPLATES authored

### Enforcement Expansion

- [x] **ENFR-02**: 32 ESLint AST rules for code structure and hooks
- [x] **ENFR-03**: 64 regex rules for banned strings, imports, naming
- [x] **ENFR-04**: Every pattern tracked across all 7 enforcement mechanisms
- [x] **ENFR-05**: Pattern staleness resolution — enforced, documented, or
      removed
- [x] **ENFR-06**: Coverage accurately calculated at 17.2% (target revised from

- [x] **ENFR-07**: FP auto-disable for rules above threshold

### Health Monitoring & Dashboard

- [x] **HLTH-02**: 64-metric composite scoring across 8 categories
- [x] **HLTH-03**: ecosystem-health-log.jsonl persists scores for trending
- [x] **HLTH-04**: Interactive /ecosystem-health dashboard with 13 dimensions
- [x] **HLTH-05**: Warning lifecycle system with JSONL persistence
- [x] **HLTH-06**: Mid-session alerts for metric degradation

### Gate Recalibration & Automation

- [x] **GATE-03**: Override analytics with pattern detection
- [x] **GATE-04**: Auto-archive reviews with session safety
