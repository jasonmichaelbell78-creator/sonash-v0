**Dual-form decision** (locked, SWS re-eval 2026-03-14): T3 is both a CANON
tenet and a standalone `/convergence-loop` skill. Tenet defines WHEN/WHY (any
investigation producing claims where wrong claims waste effort). Skill defines
HOW (agent prompts, tally formats, convergence logic, domain slicing).

**Tenet summary (T3/T21):** Research requires convergence loops, not single
passes. Min 2 passes. Each pass: verify prior claims + expand new angles.
Explicit tally (confirmed/corrected/extended/new). Convergence = no material
corrections + no unexplored angles. All corrections must cite evidence.

**Architecture:** One skill, many callers. High-value: deep-plan Phase 0, audit
skills, code review, /systematic-debugging, doc-code sync. Medium-value: test
generation, migration verification, PR review processing.
