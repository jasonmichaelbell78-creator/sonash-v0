## If you need help right now (crisis / urgent support)

- **Emergency:** Dial **911**
- **988 Suicide & Crisis Lifeline (call/text/chat 24/7):**
  [988lifeline.org](https://988lifeline.org/)
- **Tennessee Statewide Crisis Line (24/7):**
  [855-274-7471 (855-CRISIS-1)](https://www.tn.gov/health/health-program-areas/fhw/suicide-prevention/how-to-get-help.html)
- **Crisis Text Line (TN):** Text **TN** to **741741** —
  [Crisis Text Line info (TN DOH)](https://www.tn.gov/health/health-program-areas/fhw/suicide-prevention/how-to-get-help.html)
- **Mental Health Cooperative (Nashville) — Crisis services:**
  - [Emergency Psychiatric Services / Walk-in Crisis Treatment Center](https://www.mhc-tn.org/our-services/emergency-psychiatric-services/)
  - [Contact / crisis numbers](https://www.mhc-tn.org/contact-us/)
