## Regeneration

To regenerate archived pipeline intermediates, run the full TDMS pipeline:

```bash
node scripts/debt/extract-audits.js
node scripts/debt/extract-sonarcloud.js
node scripts/debt/extract-reviews.js
node scripts/debt/normalize-all.js
node scripts/debt/dedup-multi-pass.js
node scripts/debt/generate-views.js
```
