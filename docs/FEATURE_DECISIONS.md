## How to Use

1. Review each feature
2. Check the appropriate box:
   - `[x]` = **Add to roadmap**
   - `[-]` = **Defer** (maybe later)
   - `[~]` = **Reject** (not for us)
3. Add any notes in the Comments column
4. When back home, share this with Antigravity to implement decisions
